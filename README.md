# コラッツ予想シミュレーター

任意の正の整数に対してコラッツ予想の数列を計算し、表示するツールです。

## コラッツ予想とは

次のルールを繰り返すと、どんな正の整数から始めても必ず 1 に到達するという予想です。

- **偶数**のとき → `n / 2`
- **奇数**のとき → `3n + 1`

例: 6 → 3 → 10 → 5 → 16 → 8 → 4 → 2 → 1（8ステップ）

## 必要環境

- Python 3.10 以上
- [uv](https://github.com/astral-sh/uv)

## 使い方

### 対話モード

```bash
uv run collatz.py
```

```
正の整数を入力してください: 27
開始値: 27
ステップ数: 111
最大値: 9232
数列: 27 → 82 → 41 → ...
```

### コマンドライン引数

```bash
# 1つ指定
uv run collatz.py 27

# 複数まとめて指定
uv run collatz.py 6 12 100
```

## 出力項目

| 項目 | 説明 |
|------|------|
| 開始値 | 入力した正の整数 |
| ステップ数 | 1 に到達するまでの操作回数 |
| 最大値 | 数列中の最大値 |
| 数列 | 全ステップの数値を `→` でつないだもの |

## テスト

```bash
uv run pytest test_collatz.py -v
```
